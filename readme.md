## Requirements

- Go 1.16 or later

## Installation

1. Clone the repository:

    ```sh
    git clone https://github.com/amsteffensen/goapp.git
    cd goapp
    ```

2. Install the dependencies:

    ```sh
    go mod download
    ```

## Usage

1. Run the application:

    ```sh
    go run main.go
    ```

2. Open your web browser and go to `http://localhost:8080`.
